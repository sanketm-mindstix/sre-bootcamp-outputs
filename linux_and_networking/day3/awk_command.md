## awk command
`awk` is a **pattern scanning and text processing tool**.  
It reads input **line by line**, splits each line into **fields (columns)**, and lets you apply logic.


| Symbol | Meaning |
|--------|--------|
| `$0` | Entire line |
| `$1` | First column |
| `$2` | Second column |
| `$NF` | Last column |
| `NF` | Number of fields |
| `NR` | Current line number |
| `FS` | Field separator |
| `OFS` | Output field separator |
